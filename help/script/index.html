Script bindings reference

This section provides a complete reference to the Qt Script bindings that expose Qt C++ APIs to the QtScript runtime.

How to use this reference

  • Getting started - run your first script and learn the basic conventions.
  • Packages - discover the high-level modules that group related functionality (qt.core, qt.gui, qt.network, ...).
  • All classes - browse every wrapped Qt class, its constructors, properties, methods, signals and static functions.
If you are new to QtScript itself, skim the official Qt Script documentation first - this reference focuses only on what becomes available after the bindings are loaded.
